# Tech Stock Behavior Analysis & Prediction (2014-2024)

## 📌 Project Overview
This project examines the stock price behaviors and returns of four major technology companies—**Apple, Google, Microsoft, and Amazon**—over a ten-year period. The study utilizes statistical methods and **Long Short-Term Memory (LSTM)** neural networks to evaluate past trends and predict future price movements.

## 📁 Repository Structure
* **Part_1_EDA/**: Contains notebooks for historical price views, volume of sales, moving averages, and correlation analysis.
* **Part_2_Models/**: Contains individual Deep Learning (LSTM) implementation and evaluation for each tech stock.
* **Documentation/**: Includes the full 60-page Final Year Project (FYP) academic research report.
* **Datasets**: Original CSV files for Apple, Google, Amazon, and Microsoft stock data from Kaggle.

## 💡 Key Research Findings
* **Highest Accuracy:** Predictive models demonstrated an overall upward trend, with **Google (GOOGL)** showing the highest prediction accuracy.
* **Volatility:** **Microsoft (MSFT)** exhibited the most significant volatility during the study period.
* **Risk & Return:** **Amazon (AMZN)** presented the highest risk but also the highest potential return.
* **Correlations:** Analysis revealed strong positive correlations among the four technology companies.

## 🛠️ Skills Demonstrated
* **Machine Learning:** Long Short-Term Memory (LSTM) Networks, KDD Process, Data Normalization, and Feature Scaling.
* **Data Science:** Financial Time-Series Analysis, Risk vs. Return Profiles, and Predictive Modeling.
* **Technical Tools:** Python (TensorFlow/Keras, Pandas, NumPy), SQL, and Data Visualization (Matplotlib, Seaborn).

## 📑 Methodology (KDD Process)
The project follows the **Knowledge Discovery in Database (KDD)** phases:
1. **Data Collection & Cleaning**.
2. **Data Normalization & Transformation**.
3. **LSTM Model Training & Parameter Tuning**.
4. **Performance Evaluation** (Using RMSE, MAE, and MSE).
